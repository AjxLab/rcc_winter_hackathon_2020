# voice similarity

音声の類似度を算出

## 概要
### 処理の流れ
#### MFCCを利用
- 比較対象となる２つの音声を用意
- 音声をリサイズ・リサンプリング（可変長の音声を調整するため）
- 音声をMFCC（メル周波数ケプストラム係数）に変換
- それぞれの音声のMFCCの決定係数を類似度とする
#### LPCを利用
- 100~1000Hzのバンドパスフィルタをかける
- LPC分析によってスペクトル包絡を求める
- 極大値・極小値を求める
- それぞれの音声の極大値・極小値のコサイン類似度を類似度とする
### キャラクター一覧
- 大蛇丸「潜影蛇手」
- ドラえもん「僕ドラえもん」
- しずかちゃん「のび太さーん」
- 悟空「オッス！オラ悟空」
- フリーザ「私の戦闘力は53万です」
